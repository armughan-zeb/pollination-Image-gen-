# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-01-12

### Added
- Initial release of Pollination Image Generator
- Support for Flux and Turbo AI models
- Multiple image dimension options
- Download functionality for generated images
- Copy URL to clipboard feature
- Modern dark-themed UI with glassmorphism effects
- Responsive design for mobile and desktop
- Loading animations and visual feedback

### Features
- Real-time image generation via Pollinations AI API
- Model selection dropdown
- Custom dimension presets
- One-click download button
- URL copy functionality

## [Unreleased]
- Additional AI models support
- Image history feature
- Prompt suggestions
