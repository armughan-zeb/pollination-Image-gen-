# Getting Started

## Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required!

## Quick Start

1. **Download or Clone**
   ```bash
   git clone https://github.com/armughan-zeb/pollination-Image-gen-.git
   cd pollination-Image-gen-
   ```

2. **Open in Browser**
   - Simply double-click `index.html`
   - Or use a local server: `npx serve .`

3. **Generate Your First Image**
   - Enter a prompt describing your image
   - Select a model (Flux or Turbo)
   - Choose dimensions
   - Click "Generate Image"

## Tips for Better Results

### Writing Good Prompts
- Be specific and descriptive
- Include style keywords (realistic, cartoon, oil painting)
- Mention lighting and mood
- Add details about colors and composition

### Example Prompts
- "A futuristic city at sunset with flying cars, cyberpunk style"
- "A cozy cabin in the mountains during winter, watercolor painting"
- "Portrait of a cat wearing a space suit, photorealistic"

## Troubleshooting

### Image Not Loading?
- Check your internet connection
- Try a different browser
- Wait a moment and try again

### Slow Generation?
- The Flux model may take longer for complex prompts
- Try the Turbo model for faster results
