# Frequently Asked Questions

## General Questions

### What is Pollination Image Generator?
A free, open-source AI image generation tool that uses the Pollinations AI API to create images from text prompts.

### Is it free to use?
Yes! The application is completely free. It uses the Pollinations AI API which is also free to use.

### Do I need to create an account?
No account or API key is required. Just open the app and start generating!

## Technical Questions

### What browsers are supported?
All modern browsers including:
- Google Chrome (recommended)
- Mozilla Firefox
- Microsoft Edge
- Safari

### Can I use this offline?
No, an internet connection is required to communicate with the Pollinations AI API.

### How long does image generation take?
- **Turbo model**: 5-15 seconds
- **Flux model**: 15-30 seconds

Generation time may vary based on server load and prompt complexity.

## Image Questions

### What dimensions are available?
- 512x512 (Square)
- 768x768 (Square HD)
- 1024x768 (Landscape)
- 768x1024 (Portrait)
- 1024x1024 (Square Ultra)

### Can I use generated images commercially?
Please refer to [Pollinations AI's terms of service](https://pollinations.ai) for usage rights.

### How do I get better results?
See our [Getting Started Guide](docs/GETTING_STARTED.md) for tips on writing effective prompts.

## Troubleshooting

### The image won't generate
1. Check your internet connection
2. Try refreshing the page
3. Try a different browser
4. Wait a moment and try again

### The download button doesn't work
Try right-clicking the image and selecting "Save image as..."
